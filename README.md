# Hermes909 Autonomous AI Agent

Hermes909 is a Docker-based autonomous AI agent environment powered by Hermes-Agent with OpenRouter cloud inference.

## Features

- Dockerized AI agent runtime
- OpenRouter cloud model integration
- Discord gateway support
- Persistent Hermes memory/config
- Browser automation ready
- Playwright support
- Coding & autonomous task execution
- Auto-restart container support

## Stack

- Hermes-Agent
- Docker Compose
- OpenRouter API
- Ubuntu 22.04
- Node.js
- Python3
- Playwright
